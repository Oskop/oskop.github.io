Tipe Data

Untuk membuat sebuah program, kita harus selektif dalam memilih tipe data yang tepat untuk tiap variabel yang kita buat. Tujuan dari menentukan tipe data yang pas adalah untuk mengehmat memory yang dibutuhkan dan membuat program bejalan lebih cepat.

Tipe data dibagi menjadi 2, yaitu tipe data primitive dan non-primitive. Kedua jenis tersebut juga bergantung pada bahasa pemrograman yang digunakan.

  • Boolean merupakan tipe data yang hanya berisi satu bit nilai baik itu true atau pun false. Ukuran data untuk tipe boolean yang ditampung di dalam memory (RAM) tergantung dari komputer atau komputer virtual (Virtualization, seperti halnya OS di dalam). 

    Java
boolean t = true;
boolean f = false;

Python
t = True
f = False

  • Byte adalah tipe data angka berukuran 1 Byte (8-bit) dengan nilai minimum -128 (-27) dan nilai maksimum 127 (27 – 1). Hanya berupa bilangan bulat dan sangat cocok sekali untuk menghemat memory RAM. 

    Java
byte a = 50;
byte c = -128;

  • Short adalah tipe data dengan dengan ukuran 2 Byte (16-bit) dengan nilai minimum -32,768 (-215) dan nilai maksimum 32,767 (215 – 1). Hanya berupa angka bilangan bulat dan bisa untuk menghemat memory RAM. 

    Java
short min = -300
short max = 50000

  • Int (Integer) adalah tipe data berukuran 4 Byte (32-bit) dengan nilai minimum -2.147.483.648 dan nilai maksimum 2.147.483.647. int merupakan bilangan bulat jadi segala angka di belakang koma akan dibulatkan sesuai aturan pembulatan.

    Java
int nilai = 1000000
int ialin = -1230490

Python
nilai = 12893
nilai = -89343

  • Long adalah tipe data berukuran 8 Byte (64-bit) dengan rentang nilai dari -9,223,372,036,854,775,808 sampai 9.223.372.036.854.775.807. long jugamerupakan bilangan bulat jadi segala angka di belakang koma akan dibulatkan sesuai aturan pembulatan. 

    Java
long ln = 3898738L
long nl = -9098380L

  • Float adalah tipe data untuk bilangan desimal yang batasnya sampai 7 bilangan desimal (7 angka di belakang koma, selebihnya akan dibulatkan). Ukuran data sebesar 4 Byte (32-bit). 

    Java
float pof = 9.4533f
float opf = -9.8393f

Python
flt = 5.0
flt = -487.983

  • Double adalah tipe data untuk bilangan desimal dengan batas sampai 16 bilangan desimal (16 angka di belakang koma, selebihnya akan dibulatkan). Ukuran datanya sebesar 8 Byte (64-bit). 

    Java
double db = 8934.987387843
double bd = 79384.834789854

  • Char adalah tipe data yang hanya menampung satu Unicode saja (satu karakter baik simbol, huruf maupun angka). Ukuran data sebesar 2 Byte (16-bit). Rentang Unicode antara '\u0000' (0) sampai '\uffff' (65535). (silahkan browsing unicode itu apa, bagi yang belum paham) 

    Java
char karakter = 'H'

Kalau yang lain bisa, berarti sudah mainstream.
