Program Pengubah Nilai Suhu Celcius ke Lainnya

Yap. Kali ini kita akan mencoba membuat program sederhana yang mengubah nilai suhu dari satuan derajat ke satuan lain, seperti Farenheit, Reamur dan Kelvin.

Perlu kalian ketahui bahwa dalam praktek ini, kalian harus paham rumus pengubahan dari celcius ke satuan lain.
Celcius ke Farenheit = ( suhu_celcius * (9.0 / 5.0) ) - 32
Celcius ke Reamur = suhu_celcius * (4.0 / 5.0)
Celcius ke Kelvin = suhu_celcius + 273.15

Setelah mengetahui cara mengubah nilai satuan di atas, selanjutnya kita buat dulu alur program yang dapat dilihat pada gambar di bawah ini.

Setelah membuat alur programnya, ini saatnya menulis code dengan bahasa pemrograman (Kalian bisa jalankan code di bawah dengan klik link tiap segmen).

[Python] 
                                            
celcius = float(input('Masukkan suhu dalam satuan derajat celcius')) 
opsi = input(''' 
========================= 
Opsi Ubah nilai 
f / F   untuk Farenheit 
r / R   untuk Reamur 
k / K   untuk Kelvin 
========================= 
Pilih opsi: 
''')
hasil = 0 

if opsi == 'f' or opsi == 'F': 
    hasil = ( celcius * (9/5) ) + 32 
elif opsi == 'r' or opsi == 'R': 
    hasil = celcius * (4/5) 
elif opsi == 'k' or opsi == 'K': 
    hasil = celcius + 273.15 
else: 
    print('Karena anda tidak memasukkan opsi di atas, maka selesai saja lah...') 

print('Nilai setelah diubah = ', hasil, opsi.upper())

Dan untuk bahasa pemrograman Java di bawah ini

[Java] 
    
import java.util.Scanner;

public class Program
{
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Masukkan nilai suhu dalam celcius: ");
        float i_c = sc.nextFloat();
        System.out.println("==========================\n"
        + "Opsi:\n"
        + "f / F [Farenheit]\n"
        + "r / R [Reamur]\n"
        + "k / K [Kelvin]\n"
        + "==========================\n");
        System.out.print("Pilih opsi: ");
        String opsi = sc.next();
        float hasil;
        if (opsi.contentEquals("f") || opsi.contentEquals("F")) {
            hasil = (i_c * 9 / 5) + 32;
            System.out.println("\nNilai dalam Farenheit = " + hasil);
        } else if (opsi.contentEquals("r") || opsi.contentEquals("R")) {
            hasil = i_c * 4 / 5;
            System.out.println("\nNilai dalam Reamur = " + hasil);
        } else if (opsi.contentEquals("k") || opsi.contentEquals("K")) {
            hasil = (float) (i_c + 273.15);
            System.out.println("\nNilai dalam Kelvin = " + hasil);
        } else {
            System.out.println("\nDahlah selesai. Dikasih opsi malah plih yang lain.");
        }
    }
}

Pada code di atas, program akan meminta input berupa nilai suhu dalam satuan derajat celcius, kemudian program meminta input berupa opsi tujuan satuan baik Farenheit, Reamur atau Kelvin dengan memasukkan inisial nama satuan. Setelah itu, akan muncul hasil perubahan nilainya. Bila user memasukkan nilai celcius berupa huruf, maka akan terjadi error. Dan bila user memasukkan opsi selain inisial nama satuan, maka program akan langsung selesai.

Kalau tidak ada yang bertanya, ya sudah, selesai.
